Open Annotations Framework

Description

Callimachus can allow for the simple annotation of any resource (document or image) with a URL via drag and drop and iFrames. This annotation follows the Open Annotation data model, via one of six Open Annotation motivations:

  • Bookmarking: The most simple type of annotation, saving a Web URL for later reference.
  • Tagging: Via plain textual tags or semantic (URL) tags.
  • Classification: Association of RDF Concepts with a Web resource.
  • Commenting: Arbitrary comments on a Web resource.
  • Identifying: Associating one URL with another (e.g. identifying a location with a Wikipedia page for it).
  • Highlighting: Saving a selection of text from a Web page and optionally commenting upon it. Highlighting only works for resources local to Callimachus (for now).
