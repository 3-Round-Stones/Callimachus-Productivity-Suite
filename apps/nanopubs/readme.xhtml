NanoPublications in Callimachus

Description

3 Round Stones is currently exploring the implementation of NanoPublications within Callimachus. This would allow users the opportunity to create NanoPublications via a friendly user-interface without having to write raw RDF. These NanoPublications are generated via controlled vocabularies which can be loaded from existing files or created within Callimachus.

The NanoPublications can also be annotated in accordance with the Open Annotation model. Users can make arbitary annotations about these NanoPublications in the form of bookmarks, comments, tags, classifications, or descriptions. Since all of these annotations (and the NanoPublication itself) are stored as RDF behind the scenes, they can be easily queried to generate reports or analytics.

You can also try creating one yourself.
